Githubの使い方

Githubとは

Githubは自分の書いたコードを公開し、管理するツールです。公開するときには[private]か[public]を選ぶことができます。

用語

次に、Githubを使う時に必要な用語がいくつかあります。

  • リポジトリ：書いたコードを保存する場所
  • リモートリポジトリ：WEB上
  • ローカルリポジトリ：自分のパソコン上
  • コミット：一時的にパソコンに保存
  • プッシュ：Githubにあげること

プロジェクト開始

Githubでプロジェクトを始める手順について説明します。

  • 1, Repositoriesを開く
  • 2, Newをクリック
  • 3, Repository nameを設定後、Public(公開)Private(非公開)を選ぶ
  • 4, Create Repositoryをクリック

完成！

コミットしてみよう

書いたコードをGithubにあげる方法を説明します。

① 初めてのコミット

  • git init
  • git add README.md
  • git commit -m "first commit"
  • git branch -M main
  • git remote add origin (例)https://github.com/ayn54i/howto-git.git
  • git push origin main

② 二度目以降のコミット

  • git add .
  • git commit (例)"[add] img"
  • git push origin main
